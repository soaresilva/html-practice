How do I see the differences between HTML and CSS?

  1. What is the difference between the usage of HTML and CSS?

    2. HTML is used to create content in the page, CSS is used to style and design the page.

  2. What is the newest version of HTML?

    3. HTML 5

    • html

  4. Have you checked out the HTML5 cheatsheet?

    5. I did.

  5. What is the difference between semantic and non-semantic elements?

    6. A semantic element describes its meaning to both the browser and the developer (e.g. form, table). Non-semantic elements (e.g. div, span) tell nothing about their content.

  6. Why do we prefer using semantic elements?

    7. Code becomes clearer and more accessible.

  7. Do all browsers support all of the newest HTML / CSS elements?

    8. All modern browsers support HTML5 and CSS. But older versions might not support some features.
